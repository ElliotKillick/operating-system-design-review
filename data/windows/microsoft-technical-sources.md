- [Microsoft technical documentation](https://learn.microsoft.com/en-us/docs/)
- [The MSDN Show](https://www.youtube.com/playlist?list=PLyP9WfpvWcmZwZL7Rkk59V31haOHUTHSZ)
- [Channel 9](https://walkingcat.github.io/ch9-index/)
- [Windows Insider Podcast](https://windowsinsider.libsyn.com)
- [Microsoft Dev Blogs](https://devblogs.microsoft.com/)
- [Microsoft Knowledge Base (KB) articles](https://www.betaarchive.com/wiki/index.php?title=Microsoft_KB_Archive)
- [ReactOS Wiki](https://reactos.org/wiki/Special:AllPages)

- Unofficial Usenet newsgroups
